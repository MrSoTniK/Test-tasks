Реализовать консольную программу на языке C#.

При запуске программа в консоль должна вывести строку приветствия “Enter text line and press Enter for new line. Press Ctrl + S to Save file”.

Далее программа должна ожидать ввода с клавиатуры строки. По нажатию на Enter в консоли должна переводиться строка.

По нажатию комбинации Ctrl + S программа должна сохранять весь введённый текст в текстовый файл, расположенный в той же папке, где находится исполняемый файл программы.

Название текстового файла должно быть сформировано по шаблону DD-MM-YYYY-HH-mm-ss.txt, где DD - день месяца в двузначном формате, MM - номер месяца в двузначном формате, YYYY - год в четырёхзначном формате, HH - час в формате 24 часов, mm - минута в двухзначном формате, ss - секунда в двухзначном формате.

После сохранения файла программа должна вывести в консоль строку “File successfully saved. X bytes”, где X - количество байт, сохранённое в файл (размер файла на диске).
