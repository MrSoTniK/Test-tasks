Комментарий:
Для подключения к PostgreSQL использовал библиотеку Npgsql. Также для построение структуры БД была использована библиотека
технологии EntityFramewok (EntityFrameworkCore конкретно для работы с PostgreSQL). Так как по заданию таблица имеет сложный
мультизаголовок, который не поддерживается в стандартном элементе DataGrid, пришлось использовать элемент syncfusion sfDataGrid,
созданный одноименной компанией. Фильтрация и сортировка присутствует в интерфейсе данного элемента 
(https://www.youtube.com/watch?v=1t4nXyDA9l0&t=30s). Не нашел в sfDataGrid сравнения по вычету строк, поэтому сделал отдельное
окно с рассчетом данного процесса по нажатию кнопки "Сравнение". Удаление строки происходит по заполнению текстового блока с
названием "ID строки:" по ID строки таблицы. Добавление строки происходит по нажатию кнопки "Добавить строку", где заполняются
соответствующие текстовый блоки. Загрузка с txt данных происходит по нажатию одноименной кнопки, где открывается окно проводника
Windows с выбором файла для загрузки данных. Не совсем понятно, как считывать отфильтрованные/отсортированные данные с sfDataGrid,
поэтому для выгрузки было сделано отдельное окно, с фильтрацией по 5 выбираемым параметрам и с выбором столбцов для выгрузки.

Были использованы:
Visual Studio 2019, .NET Framework 4.7.2, PostgreSQL 13.2
